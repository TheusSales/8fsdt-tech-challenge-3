# 8fsdt-tech-challenge-3

Front-end do Tech Challenge 3 — React + TypeScript + Vite + Styled Components.

## Stack

- React 18 (hooks e componentes funcionais)
- TypeScript
- Vite 5
- Styled Components 6

## Pré-requisitos

- Node.js 18+
- npm

## Como rodar

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — sobe o servidor de desenvolvimento
- `npm run build` — gera build de produção
- `npm run preview` — pré-visualiza o build

## Estrutura

```
src/
  components/   # componentes reutilizáveis
  pages/        # páginas
  hooks/        # hooks customizados
  services/     # integração com a API REST do back-end
  styles/       # tema e estilos globais (Styled Components)
  types/        # tipos compartilhados
```
